# 서초구 · 서울특별시

> 서초구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-seocho.github.io/**

서초구는 옛 지명 '서리풀'에서 이름을 얻은 서울 한강 이남 중앙부의 자치구로, 대법원·대검찰청·서울중앙지방법원이 한데 모인 법조타운과 삼성전자 서초사옥, 강남역 상권이 공존하는 곳입니다. 예술의전당과 국립국악원, 한국예술종합학교가 이웃해 자리 잡아 서울을 대표하는 공연·예술 클러스터를 이룹니다. 반포동과 방배동 서래마을에는 국내 최대의 프랑스인 거주 지역이 형성되어 '서울 속 작은 프랑스'로 불립니다. 면적 약 47㎢로 서울 25개 자치구 가운데 가장 넓으며, 우면산과 청계산 자락의 녹지와 반포한강공원의 수변 경관이 도심 고밀 개발지와 어우러집니다. 서초동 법조단지, 반포·잠원의 대규모 재건축 아파트, 양재 R&D 벨트가 각기 다른 도시 표정을 만들어 냅니다.

## 주요 통계

- 면적: 47 km²
- 신설: 1988-01-01
